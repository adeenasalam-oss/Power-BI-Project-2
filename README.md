Student Academic Performance & Insight Dashboard
Power BI | Data Modeling | Business Intelligence | Data Visualization

 Project Summary
The Student Academic Performance & Insight Dashboard is an end-to-end Business Intelligence project built using Power BI.
The project analyzes student academic data to uncover performance trends and identify behavioural and socioeconomic factors influencing exam outcomes.
This solution demonstrates strong capabilities in:
Data Cleaning & Transformation
Data Modeling
DAX Calculations
Interactive Dashboard Design
Insight Generation for Decision-Making

 Business Problem
Educational institutions require actionable insights to understand:
What factors improve student exam performance?
How do attendance and study habits impact results?
Do socioeconomic conditions affect academic outcomes?
This dashboard converts raw student data into meaningful insights that support data-driven academic strategies.

 Dataset Features
The dataset includes:
Exam Score
Attendance
Hours Studied
Study Level
Motivation Level
Previous Scores
Parental Involvement
Family Income
Internet Access
Extracurricular Activities
Learning Environment

Key Analytical Insights
Increased study levels correlate with higher exam scores.
Attendance shows a strong positive relationship with performance.
Motivation significantly impacts academic results.
Socioeconomic conditions contribute to measurable performance variation.
